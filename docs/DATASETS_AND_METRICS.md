## 1. DATASETS

### 1.1 Speech Corpora

#### LibriSpeech
- **Use:** Primary dry speech for simulation + ASR evaluation
- **URL:** https://www.openslr.org/12
- **Subsets used:**
  - `train-clean-100`: Training RIR convolution (585 utterances, ~100hrs)
  - `test-clean`: Testing (87 utterances)
  - `test-other`: Hard ASR testing
- **Format:** 16kHz, 16-bit, FLAC
- **Preprocessing:** Convert to WAV. Apply VAD to get speech segment timestamps. Split into 20s chunks (matching Cross3D-Edge).
- **Ground truth:** Transcriptions in `.trans.txt` files

#### Common Voice (Mozilla)
- **Use:** Accent-robust ASR evaluation, Indian English
- **URL:** https://commonvoice.mozilla.org/
- **Preprocessing:** Resample to 16kHz if needed

#### VoxPopuli
- **Use:** Multilingual ASR evaluation
- **URL:** https://github.com/facebookresearch/voxpopuli

### 1.2 Noise Corpora

#### MUSAN (Music, Speech, and Noise)
- **Use:** Noise augmentation during simulation
- **URL:** https://www.openslr.org/17/
- **Contents:** Music (~42hrs), Speech babble (~60hrs), Environmental noise (~6hrs)
- **Preprocessing:** Segment into 10s clips. Categorize by type.
- **Usage in simulation:** Randomly select noise clip, scale to target SNR, add to simulated signal

#### DNS Challenge 2020 (Deep Noise Suppression)
- **Use:** Diverse noise types for enhancement evaluation
- **URL:** https://github.com/microsoft/DNS-Challenge
- **Contents:** 500+ hours of noise from 150 classes

### 1.3 Separation Datasets

#### WSJ0-2mix
- **Use:** Standard 2-speaker separation benchmark
- **How to create:** Mix WSJ0 utterances at random SIR from 0-5dB
- **Metrics:** SI-SDR, SDR, SIR, SAR

#### LibriMix
- **Use:** Separation evaluation based on LibriSpeech
- **URL:** https://github.com/JorisCos/LibriMix
- **Variants:** Libri2Mix (2 speakers), Libri3Mix (3 speakers), clean/noisy

#### WHAMR! (WSJ0 Hipster Ambient Mixtures with Reverb)
- **Use:** Separation in noisy reverberant conditions
- **URL:** https://wham.whisper.ai/
- **Contains:** WSJ0-2mix + real environmental noise + simulated reverb

### 1.4 SSL / Localization Datasets

#### LOCATA Challenge
- **Use:** Real-world SSL evaluation (primary real-data test)
- **URL:** https://zenodo.org/record/3630471
- **Tasks:** Task1 (static source, static array), Task3 (moving source, static array)
- **Array:** 12-mic Robot Head
- **Room:** 7.1×9.8×3.0m, RT60≈0.55s
- **Ground truth:** 3D source positions from OptiTrack motion capture
- **Metric:** MAE on azimuth and elevation

#### Synthesized Cross3D Dataset
- **Use:** Training and controlled evaluation of SSL models
- **Generated by:** gpuRIR / pyroomacoustics (see ACOUSTIC_LAB.md)
- **Parameters:** Cross3D-Edge paper Section 4.1

### 1.5 Diarization Datasets

#### AMI Meeting Corpus
- **Use:** Speaker diarization evaluation
- **URL:** https://groups.inf.ed.ac.uk/ami/corpus/
- **Contents:** 100+ hours of real meeting recordings
- **Annotations:** Speaker-level timestamps, overlap annotations
- **Metric:** DER

#### CallHome
- **Use:** Telephone diarization benchmark

### 1.6 Room Impulse Response Datasets

## 2. DATA PREPROCESSING PIPELINE

```python
# Standard preprocessing for all audio
def preprocess_audio(path, target_sr=16000):
    """
    1. Load audio file
    2. Convert to mono if stereo (for dry signals only)
    3. Resample to target_sr
    4. Normalize to [-1, 1] range
    5. Trim silence from beginning and end
    """
    audio, sr = soundfile.read(path)
    if audio.ndim > 1:
        audio = audio.mean(axis=1)  # mono
    if sr != target_sr:
        audio = librosa.resample(audio, orig_sr=sr, target_sr=target_sr)
    audio = audio / (np.max(np.abs(audio)) + 1e-8)
    audio, _ = librosa.effects.trim(audio, top_db=30)
    return audio
```

## 3. METRICS — COMPLETE REFERENCE

### 3.1 Localization Metrics

#### Root Mean Square Angular Error (RMSAE)
```python
def angular_error(doa_true, doa_est):
    """
    doa_true, doa_est: (azimuth, elevation) in radians
    Returns: angular error in degrees
    """
    az_t, el_t = doa_true
    az_e, el_e = doa_est
    cos_err = (np.sin(el_t)*np.sin(el_e) + 
               np.cos(el_t)*np.cos(el_e)*np.cos(az_t - az_e))
    cos_err = np.clip(cos_err, -1.0, 1.0)
    return np.degrees(np.arccos(cos_err))

def rmsae(doa_true_sequence, doa_est_sequence):
    errors = [angular_error(t, e) for t, e in zip(doa_true_sequence, doa_est_sequence)]
    return np.sqrt(np.mean(np.array(errors)**2))
```

#### Mean Angular Error (MAE)
```python
def mae(doa_true_sequence, doa_est_sequence):
    errors = [angular_error(t, e) for t, e in zip(doa_true_sequence, doa_est_sequence)]
    return np.mean(errors)
```

#### SI-SDR (Scale-Invariant Signal-to-Distortion Ratio)
```python
def si_sdr(reference, estimation):
    """
    reference, estimation: 1D numpy arrays, same length
    Returns: SI-SDR in dB (higher = better)
    """
    ref = reference - np.mean(reference)
    est = estimation - np.mean(estimation)
    s_target = np.dot(est, ref) / np.dot(ref, ref) * ref
    e_noise = est - s_target
    return 10 * np.log10(np.dot(s_target, s_target) / (np.dot(e_noise, e_noise) + 1e-8))

def si_sdri(reference, estimation, mixture):
    """Improvement over unprocessed mixture."""
    return si_sdr(reference, estimation) - si_sdr(reference, mixture)
```

### 3.5 System Metrics

#### Real-Time Factor (RTF)
```python
def rtf(processing_time_seconds, audio_duration_seconds):
    return processing_time_seconds / audio_duration_seconds
    # RTF < 1.0 means real-time capable
```

#### Latency
```python
import time
def measure_latency(module, data, n_runs=100, warmup=10):
    for _ in range(warmup):
        module.process(data)
    times = []
    for _ in range(n_runs):
        start = time.perf_counter_ns()
        module.process(data)
        end = time.perf_counter_ns()
        times.append((end - start) / 1e6)  # ms
    return np.median(times)
```

## 4. METRIC INTERPRETATION GUIDE

When reporting results, use these benchmarks for context:

| Metric | Poor | Acceptable | Good | Excellent |
|--------|------|-----------|------|-----------|
| RMSAE (°) | >20 | 10-20 | 5-10 | <5 |
| PESQ | <2.0 | 2.0-2.5 | 2.5-3.5 | >3.5 |
| STOI | <0.7 | 0.7-0.8 | 0.8-0.9 | >0.9 |
| SI-SDRi (dB) | <5 | 5-10 | 10-15 | >15 |
| WER (%) | >30 | 15-30 | 5-15 | <5 |
| DER (%) | >25 | 15-25 | 10-15 | <10 |
| RTF | >2.0 | 1.0-2.0 | 0.5-1.0 | <0.5 |
